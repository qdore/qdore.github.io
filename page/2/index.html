磁盘文件用户管理

du找出文件大小

  • du file1,file2… : 找出文件占用的磁盘空间

  • du -a DIRECTORY : 递归显示文件占用的磁盘空间

  • du -h file : 显示文件大小单位(b指定b,k指定KB,m指定MB,B指定块)

  • du -c file1 file2… : 显示总文件大小

  • du -s file1 file2… : 只输出合计数据

  • du —exclude file1 file2 file3 : 排除文件

  • du —maxdepth 1 -a DIRECTORY : 指定最大深度

  • du -h : 磁盘可用空间信息

计算命令时间

  • time COMMAND : 测试运行时间

用户信息

  • who : 显示当前用户的相关信息

  • w : 显示详细信息

  • users : 显示用户列表

  • uptime : 运行时间

  • last : 上一次启动以及用户登录的信息

  • last USER / lastb / last reboot

watch监视命令输出

  • watch command : 每两秒更新一次输出

  • watch -n 5 command : 每5秒更新一次输出

  • watch -d command : 将差异用不同颜色标注

Web

web页面下载(mac原生没有)

  • wget http://www.baidu.com

  • wget —mirror www.baidu.com : 复制网站镜像

  • wget -t 5 URL : 指定重试次数5次,0为不停的重试

  • wget —limit-rate 20k URL : 限速

  • wget -Q 20m URL : 指定下载限额

  • wget -c URL : 断点续传

  • wget —user username —password pass URL : 提供认证信息,可改为—ask-password

  • lynx -dump URL : 纯文本的方式下载URL

curl基础

  • curl URL : 下载内容输出到终端

  • curl —silent URL : 避免显示进度条

  • curl —progress URL : 显示进度条

  • curl URL -o file : 保存到文件

  • curl -C - URL : 断点续传

  • curl —referer referURL URL : 指定参照页

  • curl —cookie “user=;pass=“ URL : 指定cookie

  • curl -I URL : 打印http头

  • curl -u username:password URL : 指定用户信息

  • curl -d “postvar=user&postvar2=password” URL : 发送post请求并读取网站的响应

命令

xargs格式化数据

  • cat temp | xargs : 空格替换换行

  • cat temp | xargs -n 3 : 每行最多3个参数

  • cat temp | xargs ls : 给ls传递参数

  • cat temp | xargs -I {} ls -{} : 给ls传递参数并替换字符串

  • find . -name temp -maxdepth 1 -print0|xargs -0 wc -l : 和find结合使用\0做定界符并统计行数

which whereis file和whatis

  • which command : 查找某个命令的位置

  • whereis : 和which类似

  • file : 显示文件类型

  • whatis : 描述简短的命令信息
